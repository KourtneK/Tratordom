# Tratordom - E-commerce Agrícola

## Visão Geral
Um site de e-commerce estático simples em HTML/CSS/JavaScript para produtos agrícolas. O site está em português (Brasil) e possui funcionalidade de carrinho de compras.

## Estrutura do Projeto
- `index.html` - Página inicial principal com listagem de produtos e carrinho de compras
- `contatos.html` - Página de contato
- `style.css` - Estilização do site
- `script.js` - Funcionalidade do carrinho de compras (lado do cliente)
- `*.webp`, `*.png` - Imagens dos produtos

## Pilha Tecnológica
- HTML5 Puro, CSS3, JavaScript (sem frameworks)
- Servidor de arquivos estáticos via servidor HTTP Python

## Desenvolvimento
O projeto roda usando o servidor HTTP integrado do Python na porta 5000.

## Implantação (Deployment)
Configurado como uma implantação de site estático servindo a partir do diretório raiz.
